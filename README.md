# picolor
 Determine KMEANS , average axis etc. color of an Image

## Installation
`pip install picolor`

## Usage
```python
from picolor import avg_color, kmeans_dominant_color
path_or_url = "<PATH_OR_URL>"
print(avg_color(path_or_url))
print(kmeans_dominant_color(path_or_url))
```
